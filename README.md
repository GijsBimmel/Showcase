# Showcase Portfolio

Portfolio website van UI/UX Designer Gijs Bimmel.

## Overzicht

Deze portfolio website is een moderne, volledig responsieve showcase van mijn werk als UI/UX Designer. De website is gebouwd met pure HTML, CSS en JavaScript.

## Gebruik

1. Ga naar de GitHub-pagina van dit project.
2. Klik op de groene knop Code rechtsboven.
3. Klik op Download ZIP.
4. Pak het ZIP-bestand uit op je computer.
5. Je hebt nu het hele project lokaal.
   
## Project Structuur

- `index.html` - Hoofdpagina
- `work.html` - Projecten overzicht
- `about.html` - Over mij
- `services.html` - Services
- `contact.html` - Contact
- `project1-4.html` - Project detail pagina's
- `styles.css` - Alle styling
- `images/` - Afbeeldingen

## Features

- Responsive design
- Smooth scroll animaties
- Project filtering
- Mobile hamburger menu
